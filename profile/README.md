## 📄 Overview

<a href="#amphetamine">Amphetamine</a> is the most capable and configurable Mac sleep prevention application — providing precise control over when and how the Mac stays awake, with trigger conditions, session management, and automation capabilities that go far beyond the simple sleep prevention that basic keep-awake utilities provide.

The <a href="#amphetamine">session-based keep-awake system</a> defines exactly how long sleep prevention is active — starting a timed session that ends automatically after a specified duration, an indefinite session that continues until manually ended, or a session that ends when a specific condition is met. This session control prevents the common problem with basic sleep-prevention tools where the Mac is left permanently awake because the user forgot to disable the tool — Amphetamine's timed and conditional sessions terminate automatically when the keep-awake purpose has ended. The <a href="#amphetamine">app-specific triggers</a> activate sleep prevention automatically when specified applications are running — keeping the Mac awake while a download manager is active, while a backup application is running, while a video is playing in a specific application, or while any specified application is in the foreground or running in any state. Sleep prevention starts automatically when the trigger application launches and stops automatically when it quits.

The <a href="#amphetamine">drive mounting conditions</a> keep the Mac awake while specific drives are mounted — preventing sleep while an external drive is connected, while a network share is mounted, or while any specified volume is present. This condition is specifically useful for download, backup, and file transfer operations where drive activity should prevent sleep but the activity doesn't involve a specific application. The <a href="#amphetamine">screen dimming control</a> separately manages screen behavior from sleep prevention — allowing the display to dim or turn off while preventing the Mac from sleeping, or keeping the display fully bright when required for monitoring purposes.

<a href="#amphetamine">Battery threshold conditions</a> deactivate sleep prevention when the battery drops below a specified level — preventing keep-awake sessions from draining the battery on laptops beyond the acceptable threshold. <a href="#amphetamine">Hot corners integration</a> activates or deactivates Amphetamine sessions from any configured hot corner — the quick interface method for temporarily enabling or disabling keep-awake without reaching for the menu bar.

<a href="#amphetamine">AppleScript automation</a> controls Amphetamine from AppleScript and automation workflows — starting and stopping sessions, checking current status, and integrating Amphetamine's sleep prevention into larger automation sequences. <a href="#amphetamine">Free Mac App Store availability</a> — Amphetamine is completely free with all features available at no cost, supported through a tip system for users who want to support the developer without requiring payment.

## 🙋 FAQ

<details>
<summary>Is Amphetamine free?</summary>

Yes. Amphetamine is completely free from the Mac App Store with all features available at no cost. A voluntary tip option supports the developer.

</details>

<details>
<summary>How does Amphetamine prevent sleep?</summary>

Amphetamine uses macOS APIs to tell the system that sleep is not permitted for the duration of an active session — the same approach that video players and other applications use to prevent sleep during active use.

</details>

<details>
<summary>Can Amphetamine keep the Mac awake while a specific app is running?</summary>

Yes. App-specific triggers start sleep prevention automatically when a specified application launches and stop when it quits.

</details>

<details>
<summary>Does Amphetamine work with scheduled sessions?</summary>

Yes. Timed sessions run for a specified duration and end automatically. Sessions also configure with specific end conditions.

</details>

<details>
<summary>Can Amphetamine allow the screen to dim while preventing sleep?</summary>

Yes. Screen dimming and sleep prevention configure separately — the display can dim or turn off while the Mac remains awake and processing.

</details>

<details>
<summary>Does Amphetamine deactivate when battery is low?</summary>

Yes. Battery threshold conditions deactivate sessions when the battery drops below a specified percentage to protect laptop battery.

</details>

<details>
<summary>Does Amphetamine support AppleScript?</summary>

Yes. Amphetamine is controllable through AppleScript for integration with automation workflows.

</details>

<details>
<summary>Does Amphetamine work on Apple Silicon?</summary>

Yes. Current versions support Apple Silicon Macs.

</details>

<details>
<summary>Can I activate Amphetamine from a hot corner?</summary>

Yes. Hot corner triggers activate or deactivate Amphetamine sessions from any configured screen corner.

</details>
